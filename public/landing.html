API CHORE APP LANDING PAGE

Description


This app allow users to catelog their chores and keep tract of work to be done and work that has been done. Users are able to add new chores, edit those chores, mark the chores that have been completed and delete chores that they no longer need. The API holds all of the users chores and has them readyly availible when the user needs them. Please read the instructions below for more information.


Navigation

Open End Points


  • Add Chore: POST/add
  • Chore List: GET/chores
  • Edit Chore: PATCH/chores/chore_id
  • Delete Chore: DELETE/chores/chore_id
